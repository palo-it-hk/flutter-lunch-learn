Setup

Step 0: Knowledge Prerequisiteâ€‹

You should be comfortable coding in a modern high-level programming language, preferrably dart. Read the Dart Language Tour for a quickstart.

The setup also requires you to have knowledge in using terminal. Read this well-written guide about terminal for a quickstart.

Step 1: Install Flutterâ€‹

Go to Flutter Offical Install Guide, then:

  • Choose your platform
  • Follow Get the Flutter SDK.
note

Remember to update your path permanantly

At the end, you should run flutter doctor and see outputs.

Step 2: Choose your emulator and install stuffâ€‹

We need a device to test and develop our app. There are two choices:

  1. iOS (macOS only): Follow the Flutter Offical iOS Setup Guide
  • Follow Install XCode
  • Follow Set up the iOS simulator
  1. Android: Follow the Flutter Offical Android Setup Guide

  • Follow Install Android Studio

  • Follow Agree to Android Licenses

  • Follow EITHER Setup your Android device if your have an actual Android phone and want to use that for development

    OR

  • Follow Setup the Android emulator

Step 3: Choose and setup an editorâ€‹

Follow Flutter Offical Editor Guide to choose and setup and editor.

If you are unsure, choose VS Code.

Step 4: Clone the project codeâ€‹

If you are familiar with git, go to the project repository and clone the project.

If you are not familiar with git, you can download the project zip file and unzip it in your desired empty folder.

Step 5: Try to run the projectâ€‹

Open the project folder with your editor.

Then, follow the "Run the app" step in this guide for your desired editor.

DO NOT follow the "Create the app" step.
